# Rustmacs

This is my personal [Emacs](https://www.gnu.org/software/emacs/) distribution. 

## Installation

just move this directory into your `.emacs.d` directory
```sh
git clone https://github.com/natrixaeria/rustmacs
mv -uT rustmacs ~/.emacs.d
```
